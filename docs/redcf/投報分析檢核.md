# 投報分析檢核手冊（都更全案投報）

> 內部方法論。整理自多案投報 Excel 之**共通架構**（涵蓋：防災都更、一般都更＋容積移轉、大基地三軌〔全案管理·合建·買賣〕、權利變換 等型態）。
> ⚠️ 通用方法論，**不含任何真實案件金額**；數字一律以結構/比率/通則示意。版本 1.0｜2026-06。

---

## 1. 定位與邊界（與坪效分析「不打架」）

都更前期評估分兩段、**上下游串接、各司其職**：

| 層 | 範圍 | 性質 |
|---|---|---|
| **坪效分析（容積查核）** | 基地→FA→獎勵→允建容積→§162 三項免計→計入容積/容積餘量→銷售坪 | **圖說為真的 oracle**，逐層查核，黃金測試鎖定 |
| **投報分析（都更全案）** | 允建/銷售坪→總銷→共同負擔→分回/報酬率 | 接在坪效**之後**的財務層 |

**邊界鐵律**：投報分析只「**讀**」坪效分析的輸出（允建容積、銷售坪），**不回頭重算或覆蓋**容積查核邏輯。
- 改投報參數（單價、成本率）→ 不動 §162 容積。
- 改容積（樓層、獎勵、免計）→ 一律回坪效層，再讓投報引用新輸出。
- 對應工具：坪效＝`calc_容積查核`(L2–L4)、投報＝`calc_投報全案`(L6)。

## 2. 標準分頁架構（跨四案歸納）

投報 Excel 不論繁簡，骨架一致。分頁可歸為五群：

| 群 | 分頁 | 角色 | 簡版 | 完整版 |
|---|---|---|:--:|:--:|
| **A 權屬/權值** | 原建物及土地面積 | 更新前建物·土地分配 | ✓ | ✓ |
| | 清冊 / 清冊分配表 | 逐戶土地/建物/持分 | ✓ | ✓ |
| | **權值表** | 樓層 × 權值占比/權值比（權變分回基礎） | — | ✓ |
| **B 容積引擎** | **建築師坪效分析** | 容積→坪效（面積分析表）＝坪效層 | ✓ | ✓ |
| **C 投報主表** | **坪效及獲利分析(全案管理)** | 坪效→總銷→共負→投報 | ✓ | ✓ |
| | 坪效及獲利分析(合建) | 同上·建商分屋版（管理費＝總成本×%） | ✓ | ✓ |
| | 坪效及獲利分析(買賣) | 同上·買賣版 | — | ✓ |
| **D 分配** | 簡報檔清冊 / (合併列印) | 逐戶分回(委建/合建/買賣)對外 | ✓ | ✓ |
| **E 時程/費率** | 開發策略 | 合作模式比較 | ✓ | ✓ |
| | 現金流量表 | 分期現金流（規劃→基礎→結構→使照→交屋） | ✓ | ✓ |
| | **管理服務費** | 全案管理費撥付階梯 | — | ✓ |

> **判讀要訣**：先認出 **C「坪效及獲利分析」主表**（投報心臟），其餘分頁都是「餵它」或「接它」的子表。三版主表（全案管理／合建／買賣）只差在**成本分析區塊**與**管理費認列方式**，坪效與總銷區塊相同。

## 3. 各分頁連動關係（資料流）

```
 [A 原建物面積 / 清冊 / 權值表]   ← 更新前權屬、權值係數、持分
              │ 更新前權利價值、持分、樓層權值占比
              ▼
 [B 建築師坪效分析]   基地·容積率·獎勵 → 允建容積 · 總樓地板 · 公設 · 銷售坪
              │ 允建容積、銷售坪、總樓地板  ＝ 坪效層輸出（唯一真實量體）
              ▼
 [C 坪效及獲利分析(全/合/買)]   總銷 → 共同負擔六科目 → 共負比 / 分回 / 報酬率
        │                       │
  總成本/總銷 │                       │ 共負比、分回比
        ▼                       ▼
 [現金流量表]            [D 清冊分配表 → 簡報檔清冊]   逐戶權值 × 分回 → 各戶分配坪 / 找補
        ▲
 [管理服務費]   依 總銷 / 總成本 × 費率階梯撥付
```

**連動鐵則**
1. B 的「允建容積／銷售坪」是 C 的**輸入**；**C 不自己另算容積**（否則與 B、與坪效查核三方打架）。
2. C 的「總成本／總銷」回饋 → 現金流量表（分期）與 管理服務費（撥付基數）。
3. C 的「共負比／分回」× A 權值表的逐戶權值 → D 各戶分配。
4. **改任一輸入，沿箭頭往下游追**，確認 C → D、C → 現金流 全部連動。最常漏：**改了 B 的容積，C 的銷售坪沒跟著更新**。

## 4. 獲利分析主表——四大區塊逐塊查核

主表「坪效及獲利分析」橫向切四區塊，由左到右：

### ① 坪效分析（左區）
- 基準容積 FA ＝ 基地（**使照面積，非謄本**）× 容積率
- 允建容積 ＝ FA ×（1 ＋ 都更獎勵 ＋ 防災/規模 ＋ 容移率）；**獎勵率拆項要與建築師面積表一致**
- 免計三項（§162，**逐層**）：機電(允建×15%)、梯廳((允建+機電)/0.92×%)、陽台((…)×%)
- 專有 ＝（允建 ＋ 陽台 − 樓層×4.2 電梯）；總銷面積 ＝ 專有 /(1 − 公設比)
- ⚠️ **此區數字應＝坪效查核(工具 L2–L4)的輸出**；兩邊不符＝有人改了沒連動（見 §1 邊界鐵律）
- 查核：**銷坪比落 1.58–1.68**（住宅）；逐層免計**勿用 FA×% 總量法**（會把超出量放大約 10 倍）

### ② 總銷分析（中區）
- 總銷 ＝ 住宅坪×住宅單價 ＋ 店舖坪×店舖單價 ＋ 車位數×車位單價
- 查核：店舖單價 ≈ 住宅 ×1.4；車位分平面/機械不同單價；**平均單價回推 vs 市場評估報告**

### ③ 共同負擔（成本，右中區）
- 結構＝六大科目（見 §5）
- 查核：營建成本用 **總樓地板坪**（非銷售坪）；**看清各費率的「基數」**——代銷/稅是乘「總銷」、設計/工管是乘「營造」、管維是乘「工程」

### ④ 投資報酬（右區）
- 共同負擔比 ＝ 總成本 / 總銷；住戶分配比 ＝ 1 − 共負比
- 報酬率 ＝ 利潤 / 總成本；**全案管理利潤 ＝ 總銷 × 管理費率**（實施者收，不分屋）
- 查核：三版(全/合/買)報酬率邏輯差異；**共負比 ＞ 65%（分回 ＜ 35%）→ 回頭檢視地主接受度**

## 5. 共同負擔六大科目（法源對照）

《都市更新權利變換實施辦法》費用負擔分六類，即投報「成本分析」區：

| 科目 | 內容 |
|---|---|
| A 工程費用 | 拆除、營建(單價×總樓地板)、設計監造、工程管理、空汙費 |
| B 管維費用 | 容積獎勵後續管理維護(綠建築/無障礙/耐震/智慧) |
| C 權變費用 | 估價/規劃/測量/鑑界/鑽探/鄰房鑑定、**拆遷補償、租金補償(安置)**、地籍整理、審查費 |
| D 貸款利息 | 土融 ＋ 建融 ＋ 週轉金利息 |
| E 稅捐 | 營業稅、印花稅、土地增值稅 |
| F 管理費用 | 行政、信託、實施者(全案管理)服務費 |

> 共同負擔比（總成本/總銷）＝地主共同分攤比例；1 − 共負比 ＝ 住戶分回。

## 6. 跨案檢核清單（踩坑點）

開任何一份投報 Excel，逐項核：

- [ ] **公設比「顯示值 ≠ 公式實際值」**：曾見儲存格顯示 30%、公式實際用 34%。用「專有 /(1−n) ＝ 總銷面積」反推驗證。
- [ ] **費率被四捨五入藏真值**：設計費顯示 0.3 實為 0.35；代銷顯示 10% 實為「5%×折現係數」。**一律回推 `小計 ÷ 基數` 驗證**。
- [ ] **營業稅 1% vs 5%**：前期試算常以 1% 簡化，正式權變共負須 5%。**先確認版本別**。
- [ ] **`#REF!` / 外部連結 `[1]他檔!`**：原建物面積、現金流量表、標籤列常見斷鍵；**開檔重算數字會跑掉**，務必另做乾淨重算版核對。
- [ ] **建融利息基數是手填常數**（≈總成本，非連動公式）：用以規避循環參照；改總成本記得回填基數。
- [ ] **營建坪 vs 銷售坪 vs 允建坪**：營建成本用 **總樓地板坪**，誤用銷售坪會高估約 60%。
- [ ] **使照面積 vs 謄本面積**：FA 一律用**使照面積**；用謄本會墊高免計上限、掩蓋超容。
- [ ] **獎勵率拆項對齊**：投報「獎勵率」拆回都更/防災/規模/容移，且與建築師面積表、§65 上限對得上。
- [ ] **三來源收斂**：建築師坪效 / 代書權值 / 建設投報，三方銷售坪差 **±2%**、公設比差 **±1%** 內才算收斂。
- [ ] **逐戶分配回算**：簡報檔清冊各戶分配坪還原 ＝ 應對得上 C 的總分回；權值表占比合計 ＝ 100%。
- [ ] **車道門檻**：服務車位 ≥ 50 輛需雙車道（建築技術規則施工編 §60），影響地下效率與成本。
